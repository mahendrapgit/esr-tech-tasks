# esr-tech-tasks
## Installation guide

### Set laravel .env file and run following commands:

- php artisan migrate

- php artisan db:seed --class=UserSeeder

- php artisan storage:link

### For setting pusher
#### Set following values in .env file
- PUSHER_APP_ID=
- PUSHER_APP_KEY=
- PUSHER_APP_SECRET=
- PUSHER_APP_CLUSTER=
